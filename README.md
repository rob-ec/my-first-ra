# My First AR Project - Corinthians Symbol

This is a Web Augmented Reality (WebAR) project developed using **A-Frame** and **AR.js**. It displays an interactive 3D model of the Corinthians symbol when a specific marker is detected by the camera.

# ðŸš€ Features

- **Marker Tracking:** Uses the standard "Hiro" marker to position the 3D content.
- **Interactive Scaling:** Press and hold (mouse or touch) to temporarily increase the size of the model.
- **Interactive Animation:** Click or tap the model to trigger a 360-degree rotation.
- **Internal Animations:** Supports GLTF internal animations via `animation-mixer`.
- **Lighting:** Custom ambient and directional lighting setup.
